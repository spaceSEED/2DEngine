# 2DEngine
A simple Java-based 2d game engine, using the Java Graphics library.
For use with simple 2D game projects.

Currently Supported:
-Background layers
-Foreground layers
-Layer Scaling
-Parallax layers
-Basic Sprites
-Sprite Scaling
-Multi-key keyboard inputs
-Mouse input support


Soon-to-come:
-Sprite Animation
-Sprite Rotation
-Audio Support
-Foreground Text
